# Python Contact Management System

A simple contact management system developed using Python for managing user contacts through a command-line interface.

## Features

- Add new contacts
- Search contacts
- Update existing contacts
- Delete contacts
- Display all saved contacts
- File handling and data persistence

## Technologies Used

- Python
- File Handling
- Functions
- Lists & Dictionaries

## Purpose

This project was developed as part of academic practice to strengthen Python programming, problem-solving, and data management skills.
